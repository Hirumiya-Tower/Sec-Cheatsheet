# 解説

マルウェアの能力を自動で特定する静的解析ツール

# 基本構文
```
capa [ファイル] [オプション]
```
# 主要なオプション

### 表示の詳しさ

```-v``` 検出された機能が、プログラム内のどのアドレスにあるか表示する
```-vv``` どの命令がどのルールに合致したか、詳細情報をすべて表示する
```-q``` 概要表だけを表示する

### 出力・ルール設定

```-f <format>``` 出力形式を指定（jsonなど）
```-r <rules_path``` 独自のルールが含まれるディレクトリを指定
```-t <tag>``` 特定のタグ（ex: persistence) に関連するルールのみを実行

# 出力の読み方

### ATT&CK Tactic
MITRE ATT&CKフレームワークに基づく分類

```Discovery```  環境調査をしている
```Exfiltration``` データを外部にそうしんしようとしている
```Persistence``` 再起動後も動くように設定を書き換えている

### Capability
具体的に何ができるかを示す

```check foe debugger``` 解析を邪魔するため、デバッガの存在を確認している
```ecvrypt data using AES``` データを暗号化（ランサムウェアなど）しようとしている
```terminal connection``` 攻撃者のサーバーに繋ごうとしている
